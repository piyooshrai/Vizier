# Agent Workflows

This directory contains workflow files that help guide AI assistants when working with this codebase.

## Available Workflows

| Workflow | Description |
|----------|-------------|
| `/dev` | Start the development server |
| `/build` | Build for production |
| `/lint` | Run linting checks |
| `/test` | Run tests |
| `/new-component` | Create a new React component |
| `/new-page` | Create a new page |

## Usage

Reference workflows using slash commands like `/dev` or `/build` in your prompts.
